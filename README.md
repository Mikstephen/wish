# Love Letter Website

A personalized birthday website created with love for Rose.

## Features
- Interactive photo gallery
- Animated hearts
- Typewriter effect for letter
- Responsive design
- Personal reasons and memories

## How to Use
1. Open `index.html` in your browser
2. Click around to explore the interactive elements
3. View the photo gallery
4. Read the personalized letter

## Live Demo
You can view this website by opening the `index.html` file in any modern web browser.
